## 2024-05-24 - HTML Generation A11y and Responsiveness
**Learning:** When programmatically generating HTML UI artifacts (like directory tree indexes), it's crucial to include fundamental structural tags (`<html lang="en">`, `<meta charset="utf-8">`, `<meta name="viewport" content="width=device-width, initial-scale=1">`) in the templates. Without these, generated pages lack basic accessibility for screen readers, render poorly on mobile devices, and may display character encoding issues.
**Action:** Always verify that programmatic HTML generation includes modern viewport and accessibility meta tags by default, rather than assuming standard HTML5 boilerplate is present.
