## 2024-05-24 - HTML Directory Listing Semantic Navigation & Decorative Icons
**Learning:** When generating raw HTML pages (like directory listings), unicode icons (e.g., &#128193; for folders, &#x21B0; for parent dir) are announced poorly by screen readers. Furthermore, a bare `<ul>` lacks semantic context. Wrapping the list in `<nav>`, the page in `<main>`, and specifically using `<span aria-hidden="true">`, while adding descriptive `aria-label`s on the links, significantly improves the structural understanding and voice announcements for keyboard/screen reader users.
**Action:** Always wrap decorative or visually symbolic unicode characters in `<span aria-hidden="true">`, wrap primary navigation in a `<nav>` element, and explicitly provide `aria-label`s for links where visual icons substitute context.
